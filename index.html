cmder

Posted on 2015-06-03

Frustration

One of the most frustrating part of Microsoft Windows is the lack of a minimal, elegant and sleek terminal application unlike Linux or Mac. Windows has been stuck with the pre-historic MS DOS command prompt since the begining of time. It offers complete lack of customizability and very poor usability, and for that matter I cannot even resize the damn thing. I am sure a lot of Windows users share this frustration of mine.

Introduction

The are a number of alternative terminal applications but none of them are truely aesthetically please as the other platform counterparts, atleast I believed so… until I came across Cmder, a portable console emulator for Windows.

Cmder is a software package created out of pure frustration over the absence of nice console emulators on Windows. It is based on amazing software, and spiced up with the Monokai color scheme and a custom prompt layout. Looking sexy from the start.

Features

Some of it features include :

  • With help of the best : All the magic is happening through Conemu with enhancements from Clink.
  • Total Portability : Carry it with you on USB stick or in Cloud. Your settings, aliases and history can go anywhere you go. You will not see that ugly Windows prompt ever again. (thank goodness for that)
  • Git and others : If you decide to use the slightly bigger msysgit version, you will have all Unix commands ready in PATH so that you can git init or cat instantly on every machine. You can add custom binaries like wget, curl and git.

I mean what more could one ask for…
